A new data centre has opened inside a dolomite mine in Italy.

The Intacture project in Val di Non is now fully operational. It was built by a public-private partnership between the University of Trento and several private companies. The team used an already excavated 63,000 cubic metres of rock to create underground space for a supercomputing cluster.

This project shows how public funding can support critical AI infrastructure. It provides a model for how governments and private companies can work together to boost local business competitiveness. For those in the public sector, it highlights a way to use existing resources like mines to create stable and secure data environments.
