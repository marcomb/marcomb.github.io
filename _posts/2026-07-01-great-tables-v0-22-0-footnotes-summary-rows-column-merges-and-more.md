The Great Tables Python library recently released version 0.22.0 with several new features for data visualization.

This update adds tools for footnotes, summary rows, and column merging. It also improves image exports and LaTeX support. These features make it easier to build presentation-ready tables directly from Python DataFrames by layering formatting on top of the raw data.

For those working in the public sector, these tools help turn complex statistics into clear reports. Improved formatting for ranges, percentages, and summaries ensures that data is easy to interpret for all audiences.
